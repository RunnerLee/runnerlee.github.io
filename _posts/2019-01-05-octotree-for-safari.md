还是很中意 Safari 的字体. 那就装上 Octotree 吧.

首先把仓库克隆下来:

```
git clone git@github.com:ovity/octotree.git
```

然后因为我本地环境的原因, 就不自己构建了. 直接使用 `dist` 里.

```
cd dist
mv safari.safariextz safari.zip
xar -xf safari.zip
```
